# Benchmark — SMQ de l'alternant Coengi

Espace de travail pour analyser la démo vidéo et en tirer une roadmap.

## Où déposer le matériel

- **Transcription** : colle le texte dans `transcript.txt` (ou colle-le directement dans la conversation).
- **Captures d'écran** : dépose les images dans `captures/`, idéalement nommées par fonctionnalité
  (ex. `01-tableau-de-bord.png`, `02-plan-actions.png`, `03-audits.png`…). L'ordre n'est pas critique.

## Livrable

L'analyse comparative sera rédigée dans `../benchmark-coengi.md` :
1. Ce qu'il a et qu'on n'a pas
2. Ce qu'il a mieux fait
3. Recommandations d'intégration (valeur / effort / priorité)
